# calculadoraIMC - BMIcalculator

A simple Flutter app to calculate your Body Mass Index(BMI) or Indice de Massa Corporal(IMC) in portuguese.

## Inputs

<p>Peso(kg) - Weight</p>

<p>Altura(cm) - Height</p>
